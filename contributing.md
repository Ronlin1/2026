# Contribution guidelines

Before making a pull request, please make sure of the following:
- The opportunity you want to add does not already exist in the list
- Shorten the link with https://tinyurl.com/ if too long
- Make sure the correct link is attached respectively...
- Give a buffer date (about days or weeks to the deadline date)
- Finally, create a pull request!


Thank you! If you think there is anything to improve with the guideline, please contact me at ronlinx6@gmail.com
